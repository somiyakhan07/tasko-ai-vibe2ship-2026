Tasko AI is an AI-powered productivity assistant built for the Google AI Studio Vibe2Ship 2026 Hackathon.

It helps users intelligently manage tasks, habits, notes, calendars, schedules, and daily planning using natural language conversations powered by Google Gemini.

From organizing your day to detecting upcoming deadlines and generating personalized routines, Tasko AI keeps everything in one intelligent workspace.

# 🚀 Getting Started

Clone the repository

git clone https://github.com/somiyakhan07/tasko-ai-vibe2ship-2026.git
Go into the project

cd tasko-ai-vibe2ship-2026
Install dependencies

npm install
Create

.env.local
Add your Gemini API Key

GEMINI_API_KEY=YOUR_API_KEY
Run the application

npm run dev
